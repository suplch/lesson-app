服务器相关知识

- 什么是web服务器
- 前后端分离与耦合架构概念
    + 前后端混合开发, 例如 传统的php, jsp
    + 分离开发, JavaScript渲染页面, 通过ajax 方式访问后台数据, 服务端只提供数据接口
    + 分离的好处, 后台的服务接口可重用, 比如可以给 android, ios, web 前端提供统一的数据
