NodeJs 基本介绍

  • 服务端 javascript执行平台
  • 会NodeJS 有什么好处
    • 可以更好的掌握前端技术
    • 可以开发 web 服务器
    • 可以开发命令行工具, (比如 gulp, webpack 等打包工具就是nodejs 开发的)

环境安装(node nvm, npm)

    官网 https://nodejs.org/en/
    中文 http://nodejs.cn/api/

js文件的执行方式

    js 单线程执行
    异步任务, 同步任务

模块/包与commonjs 规范

  • 内置 (系统自带, node安装好就已经存在)
  • 第三方模块 (被安装在 node-modules 文件夹中)
    • npm init
    • npm install 模块名 --save
    • npm uninstall 模块名
  • 自定义 (项目里独立的js文件, 根据commonjs 规范定义)
